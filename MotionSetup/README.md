# Система управления двухколесным роботом

## Описание проекта
В данном проекте реализовано базовое программное обеспечение для управления двухколесным мобильным роботом на базе микроконтроллера Arduino. Были разработаны функции для всех основных видов движения платформы с дифференциальным приводом.

## Логика работы
Управление моторами осуществляется с помощью функции `move`, которая принимает направление и ШИМ-сигнал (скорость) для левого и правого двигателей. 

На базе этой функции реализованы следующие маневры:
* `forward(speed)` — движение прямо (оба мотора вращаются вперед).
* `backward(speed)` — движение назад (оба мотора вращаются назад).
* `turn_left(steepness)` — плавный поворот налево (левый мотор остановлен, правый вращается вперед).
* `turn_right(steepness)` — плавный поворот направо (правый мотор остановлен, левый вращается вперед).
* `rotate_left(speed)` — разворот на месте против часовой стрелки (левый мотор назад, правый вперед).
* `rotate_right(speed)` — разворот на месте по часовой стрелке (левый мотор вперед, правый назад).

В главном цикле `loop()` реализована последовательная демонстрация всех видов движения для тестирования системы.
